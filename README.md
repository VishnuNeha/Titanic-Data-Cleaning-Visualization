# Titanic Data Cleaning & Visualization Project

## Objective
Analyze and clean the Titanic dataset and generate visual insights.

## Dataset
Titanic passenger dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Data Cleaning Steps
- Handled missing values
- Removed duplicates
- Checked outliers
- Saved cleaned dataset

## Visualizations
- Age Distribution
- Survival by Gender
- Fare Outlier Detection
- Correlation Heatmap

## Key Findings
- Female passengers had a higher survival rate.
- Most passengers were aged between 20 and 40.
- Fare contains several outliers.
- Passenger class influenced survival chances.
